HTML Terminology & Definitions

  • HTML: HyperText Markup Language, used to create the structure of web pages.
  • Tag: An element in HTML surrounded by angle brackets that defines content (e.g., <p>).
  • Element: A complete HTML structure including the opening tag, content, and closing tag.
  • Attribute: Extra information inside a tag that modifies behavior or appearance (e.g., href, src).
  • Head: Section that contains meta-information like the title, charset, and links to stylesheets.
  • Body: Section where visible page content is written (text, images, links, etc.).
  • Anchor (<a>): Used to create hyperlinks between pages or to external sites.
  • Image (<img>): Embeds an image into the page using a source path.
  • List: HTML supports ordered (<ol>) and unordered (<ul>) lists for grouped items.
  • DOCTYPE: Declaration at the top of an HTML document indicating HTML version (e.g., <!DOCTYPE html>).
